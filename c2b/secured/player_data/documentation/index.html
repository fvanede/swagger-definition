Retrieve player data of users who logged into the PlayTech (PT) portal

The player data web service will return responses in the 200 range when the requests can be handled successfully by the backend system. An error in the 400 range will occur if the service or if the backend system detect a functional error. An error in the 500 range will mean that there is an infrastructural problem within the HC backend server landscape.

GET /players/{username}

Get data for the player with the given PT username. Service will check the validity of the combination of username and token. If valid, it will retrieve the user data. Returns an array of a single Players object.
